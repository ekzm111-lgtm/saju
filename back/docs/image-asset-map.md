# 사주명인 이미지 자산 매핑 가이드

## 1. 전체 평가

업로드된 이미지들은 전반적으로 아래 방향과 잘 맞는다.

- 딥 네이비 기반의 프리미엄 톤
- 골드 라인과 별자리 요소의 절제된 사용
- 신비롭지만 과하지 않은 분위기
- 랜딩페이지에서 통일감 있게 활용 가능한 스타일

현재 자산만으로도 메인 랜딩 1차 구성은 충분히 가능하다.

## 2. 파일별 권장 사용처

### hero.png

권장 사용 위치

- 메인 히어로 섹션 메인 배경
- 첫 화면 우측 비주얼 영역

좋은 점

- 좌측에 카피가 들어갈 여백이 충분하다
- 흐르는 빛이 역동적인 첫인상을 만든다
- 별자리와 오로라 톤이 브랜드 이미지와 잘 맞는다

권장 처리

- 데스크톱에서는 배경 이미지로 사용
- 모바일에서는 우측 상단 또는 중앙 일부만 크롭
- 텍스트 가독성을 위해 좌측에 어두운 오버레이 추가 권장

### cta.png

권장 사용 위치

- 하단 CTA 섹션 배경
- 중간 배너형 전환 섹션 배경

좋은 점

- 중앙 여백이 넓어 텍스트 배치가 쉽다
- 흐르는 골드 라인이 CTA 집중도를 높인다

권장 처리

- CTA 텍스트는 중앙 또는 좌측 배치
- 버튼 아래 여백을 넉넉히 주면 고급스럽게 보인다

### result-card.png

권장 사용 위치

- 결과 예시 섹션 샘플 카드 배경
- 실제 결과 상세 UI의 카드 베이스 참고 이미지

좋은 점

- 카드 프레임이 이미 완성형에 가깝다
- 동양적 디테일이 자연스럽다

주의점

- 하나의 큰 배경으로 쓰기보다 UI 카드 단위로 분리해서 참고하는 것이 더 적합하다
- 그대로 전체 화면에 쓰면 실무 UI와 충돌할 수 있다

권장 처리

- 실제 구현에서는 CSS 카드 스타일 가이드로 참고
- 또는 결과 예시 썸네일 이미지로 사용

### review.png

권장 사용 위치

- 후기 섹션 배경
- 후기 카드 래퍼 섹션 배경

좋은 점

- 중앙이 비어 있어 후기 카드 배치가 쉽다
- 프레임과 별자리 요소가 은은하다

권장 처리

- 전체 섹션 배경으로 깔고, 후기 카드는 반투명 또는 솔리드 카드로 올리기
- 모바일에서는 배경을 더 어둡게 눌러 텍스트 대비 확보

### service.png

권장 사용 위치

- 상품 섹션의 배경 또는 보조 비주얼
- 서비스 소개 카드 섹션의 배경 이미지

좋은 점

- review.png와 톤이 통일된다
- 서비스 카드나 가격 카드 뒤 배경으로 사용 가능하다

중요한 판단

- 이 파일은 "서비스 아이콘 세트"라기보다 "서비스 섹션용 배경"에 가깝다
- 실제 상품별 아이콘으로 바로 쓰기에는 개별 아이콘 분리가 되어 있지 않다

권장 처리

- 상품 섹션 전체 배경으로 사용
- 상품별 아이콘은 별도로 생성하거나 SVG로 제작 권장

## 3. 가장 잘 나온 자산

우선순위 평가

1. hero.png
2. cta.png
3. result-card.png
4. review.png
5. service.png

특히 hero.png는 메인 비주얼로 완성도가 높다.

## 4. 추가 제작 권장 자산

현재 빠진 자산은 아래다.

- 상품별 개별 아이콘 세트
- 모바일 히어로 전용 세로형 이미지
- 결제 섹션 또는 결과 확인용 보조 이미지

가장 필요한 추가 자산

1. 총운 / 연애운 / 재물운 / 직장운 / 궁합 / 올해의 흐름용 개별 아이콘
2. 모바일 히어로용 4:5 또는 9:16 버전

## 5. 프론트 적용 우선 가이드

1차 적용

- hero.png -> 히어로 섹션
- cta.png -> 하단 CTA
- review.png -> 후기 섹션
- result-card.png -> 결과 예시

2차 적용

- service.png -> 상품 섹션 배경
- 별도 아이콘 생성 후 상품 카드 삽입

## 6. 실무 적용 팁

- PNG 원본은 크기가 다소 크므로 웹 적용 전 압축 필요
- 가능하면 WebP 또는 AVIF로 변환
- 배경 이미지는 CSS background-image로 처리하고, 핵심 텍스트는 HTML로 올린다
- 모바일에서는 장식성이 강한 가장자리 요소가 잘릴 수 있으므로 safe area 기준으로 크롭한다
